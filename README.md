# EE533_Labs
Here are all the Lab files from Labs and homeworks of EE533 in USC Viterbi(Still building).

Lab 1

Writing a simple client and server C code on two different Linux Ubuntu Virtual Machines that enables send text with TCP.

Lab 2

Construct a 32-bit adder with Xilinx ISE 10.1 and do behavioral simulation to verify it's functionality.

Lab 3

Construct the Mini-IDS provided in the Lab instructions in Xilinx ISE 10.1 and simulate it's function.

Lab 4

1. Connect, configure and measure the performance between the nodes on NetFPGA.
2. Implement Lab 3's mini-IDS into NetFPGA.
3. Test it's performance and functionality.
